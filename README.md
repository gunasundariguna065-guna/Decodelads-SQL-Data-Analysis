
This project was developed as part of the DecodeLabs Industrial Training Program (Project 3: SQL Data Analysis).

The objective is to use SQL queries to extract meaningful insights from employee data by applying filtering, sorting, grouping, and aggregation techniques.

Project Goals

- Write SELECT queries
- Use WHERE clauses for filtering
- Sort data using ORDER BY
- Group records using GROUP BY
- Perform aggregations using COUNT, SUM, and AVG
- Generate business insights from structured data

Dataset

The dataset contains employee information including:

- Employee ID
- Name
- Role
- Department
- Salary
- Experience

Technologies Used

- Google Colab
- Python
- SQLite
- Pandas

SQL Operations Performed

SELECT Query

Retrieve all employee records.

WHERE Clause

Filter employees based on salary conditions.

ORDER BY

Sort employee salaries in descending order.

COUNT()

Calculate the total number of employees.

SUM()

Calculate the total salary expenditure.

AVG()

Calculate the average employee salary.

GROUP BY

Analyze employee distribution by role and department.
